# Getting Started with ADCE Framework

This guide walks you through your first ADCE cycle in 30 minutes.

## Prerequisites

1. **Claude Code installed** and working  
2. **Node.js 18+** for installation scripts  
3. **A broad idea** you want to turn into working software  

---

## Step 1: Installation (5 minutes)

### Install ADCE Framework
```bash
# From your project directory
npx adce-framework install
Use shaper to create a pitch for: "Build a personal finance dashboard"
Use architect to create a PRP for pitch.md
Use builder to start implementing PRP: user-dashboard
Use deployer to package and deploy user-dashboard
